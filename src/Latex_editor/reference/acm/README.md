# ACM reference assets

Drop the ACM template files into this directory:

```
reference/acm/
├── acmart.cls
├── ACM-Reference-Format.bst
├── samples/
│   ├── sample-sigconf.tex
│   ├── sample-manuscript.tex
│   ├── sample-acmsmall.tex
│   └── sample-sigconf-authordraft.tex
└── bib/
    ├── sample-base.bib
    ├── abbrev.bib
    └── acmart.bib
```

`WorkspaceManager.materialize` copies these into each new session workspace.
Get the latest from <https://www.acm.org/publications/proceedings-template>.
